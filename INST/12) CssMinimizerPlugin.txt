


CssMinimizerPlugin — это вакуумный упаковщик для твоего CSS.
Вот его задачи в трех пунктах:
Удаление "воздуха": Он вычищает все пробелы, табуляции, комментарии и переносы строк. Весь твой аккуратный CSS превращается в одну плотную строку кода.
Оптимизация правил: Плагин умнее, чем кажется. Если у тебя в коде дважды написано одно и то же свойство или пустые селекторы, он их объединит или удалит.
Было: .btn { color: red; } .btn { margin: 10px; }
Стало: .btn{color:red;margin:10px}
Уменьшение веса: Благодаря первым двум пунктам, размер CSS-файла может уменьшиться на 50–80%, что критически важно для быстрой загрузки сайта на мобильных устройствах.


1. установка 


npm install css-minimizer-webpack-plugin --save-dev 


2. Настройка в webpack.config.js


import CssMinimizerPlugin from 'css-minimizer-webpack-plugin';




  optimization: {
    // Включать минимизацию только в режиме production
    minimize: !isDev, 
    minimizer: [
      
      new CssMinimizerPlugin(),
    ],
  },
